# GitHub Pages Deployment Instructions

## Step-by-Step Guide to Host Your Reading Assistant Online

### Step 1: Create a GitHub Account (if you don't have one)
1. Go to https://github.com
2. Click "Sign up"
3. Follow the instructions to create your free account

### Step 2: Create a New Repository
1. Once logged in, click the **"+"** icon in the top right
2. Select **"New repository"**
3. Name it: `reading-assistant-app`
4. Description: "Text-to-speech reading assistant for Transforming Youth, Inc."
5. Make it **Public**
6. Check **"Add a README file"** (optional, we'll replace it)
7. Click **"Create repository"**

### Step 3: Upload Your Files
1. In your new repository, click **"uploading an existing file"** or the **"Add file"** button
2. Click **"Upload files"**
3. Drag and drop these files:
   - `index.html` (this is your reading assistant)
   - `README.md` (optional - documentation)
4. Scroll down and click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. In your repository, click **"Settings"** (top menu)
2. Scroll down the left sidebar and click **"Pages"**
3. Under "Build and deployment":
   - Source: Select **"Deploy from a branch"**
   - Branch: Select **"main"** and **"/ (root)"**
4. Click **"Save"**

### Step 5: Get Your Live URL
1. Wait 1-2 minutes for GitHub to build your site
2. Refresh the Pages settings page
3. You'll see: **"Your site is live at https://[your-username].github.io/reading-assistant-app/"**
4. Click that link to test it!

### Step 6: Share the Link
Now you can share this URL with:
- Parents
- Teachers
- Students
- Anyone at Transforming Youth, Inc.

The app will be live 24/7 and work on any device with a web browser!

---

## Your Final URL Format
```
https://[your-github-username].github.io/reading-assistant-app/
```

Example: If your username is "transformingyouth", your URL would be:
```
https://transformingyouth.github.io/reading-assistant-app/
```

---

## Updating the App Later
If you want to update the app:
1. Go to your repository
2. Click on `index.html`
3. Click the pencil icon (Edit)
4. Make your changes
5. Click "Commit changes"
6. GitHub will automatically update your live site!
