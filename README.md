## Technical Details

- Uses browser's built-in Speech Synthesis API (Google US English Female voice)
- Reading speed: 0.7 (natural pace for learning)
- Works offline - no internet connection required after initial load
- Compatible with Chrome, Safari, Edge, and Firefox

## For Parents and Teachers

This tool is designed to help children who struggle with reading by:
- Providing audio support for challenging text
- Building vocabulary through word collection
- Allowing independent practice at their own pace
- Tracking words that need extra attention
